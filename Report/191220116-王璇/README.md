## 问题

### 使用git的好处

便于记录更改的具体细节，为想回到原来版本提供了方便。

### 使用远程仓库的好处

便于协同开发，大家可以齐心协力合理分工完成一个项目。

### 在开发中使用分支的好处

版本迭代更加清晰

开发效率提升

利于代码review的实现，从而使整个团队开发更加规范，减少出错
